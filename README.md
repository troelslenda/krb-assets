# Kirby Assets

This repository serves as a temporary solution until the public version of `kirby-assets` is released. In the meantime, this is the **source of truth**.

For now the syncronization of the assets to kirby is a manual process.

## Spot Illustrations

The `spot-illustrations` folder is the single source of truth for both designers and developers. All spot illustration assets are maintained here to ensure consistency and easy access for all teams.

### Ruleset for Spot Illustrations
For detailed requirements and compliance, please refer to the ruleset defined in the Storybook for `kirby-extensions`. All new or updated illustrations must follow these rules to ensure quality and consistency.

- **Designers:** Use the SVGs in this folder as the canonical assets for design work, documentation, and handoff.
- **Developers:** Optimize and finalize the SVG files. Ensure all assets comply with the rules listed in Storybook before use or distribution.

**Note:** Do not modify or duplicate these assets outside this folder. Any updates or additions should be made here to keep the source of truth up to date for everyone.
